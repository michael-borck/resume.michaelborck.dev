<div class="workflow-section">

### How it works

```
cv-data.yml (single source of truth)
      |
      +---> Quarto ---------> PDF, HTML, Slides
      +---> Python scripts --> Quest, Terminal, Magazine, API Docs
      +---> FastAPI ---------> Live API + Swagger
      +---> MCP Server -----> AI assistant integration
      +---> Generator -------> llms.txt
```

Edit the YAML. Run `make all`. Ten formats update. [View on GitHub →](https://github.com/michael-borck/resume.borck.dev)
